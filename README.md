# React/Sass/Redux Boilerplate

Boilerplate and guide for a React/Sass/Redux.

## Getting Started

```
> npm install
```

Run an initial webpack build
```
> webpack
```

Start the development server
```
> npm run start
```

To view your project, go to: [http://localhost:3000/](http://localhost:3000/)
